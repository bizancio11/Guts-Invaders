# Imágenes opcionales del juego

Puedes poner aquí tus imágenes PNG para reemplazar los dibujos generados por código.

Nombres que carga `main.py` automáticamente:

- `player.png`: avión del jugador, se escala a `100x70` píxeles.
- `enemy_drone.png`: enemigo tipo dron, se escala a `70x42` píxeles.
- `enemy_ship.png`: enemigo tipo nave, se escala a `70x42` píxeles.
- `spike.png`: obstáculo del suelo, se escala a `54x62` píxeles.
- `bullet.png`: disparo, se escala a `24x8` píxeles.

Si falta alguna imagen o el PNG no se puede cargar, el juego usa el dibujo procedural original como respaldo.
